\section{Conclusion}

\begin{frame}{Summary}
    \begin{itemize}
        \item \textbf{The Problem:} Unobserved confounders severely bias standard regression in time series.
        \item \textbf{The DeCoR Insight:} By moving to the frequency domain, we can exploit the spectral sparsity of confounders.
        \item \textbf{The Method:} Treating confounded frequencies as adversarial outliers allows robust regression (like Torrent) to recover the true causal effect.
        \item \textbf{Our Extension:} We validated the authors' hypothesis, demonstrating that covariance-thresholding allows DeCoR to succeed even when the confounder is dense toward the target variable.
    \end{itemize}
\end{frame}
