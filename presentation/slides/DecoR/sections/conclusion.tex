\section{Conclusion \& Future Work}

\begin{frame}{Conclusion}
    \begin{itemize}
        \item \textbf{The Problem:} \textcolor{blue}{Unobserved confounders} severely bias standard regression in time series.
        \item \textbf{The DeCoR Insight:} By moving to the \textcolor{blue}{frequency domain}, we exploit the \textcolor{blue}{sparsity} of confounders.
        \item \textbf{The Method:} Treating confounded frequencies as \textcolor{blue}{adversarial outliers} allows robust regression (like \textcolor{blue}{Torrent}) to recover the true causal effect.
    \end{itemize}
\end{frame}
